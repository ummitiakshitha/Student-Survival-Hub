# 🎓 Student Survival Hub

A modern and responsive student productivity platform designed to help students manage academics, placements, productivity, mental wellness, and daily goals — all in one place.

---

# 🌟 About The Project

Student Survival Hub is a student-focused web application created to simplify student life by combining useful tools and resources into one platform.

The website helps students:

* Stay productive
* Organize daily tasks
* Prepare for placements
* Access learning resources
* Manage stress
* Track goals
* Improve consistency

The platform is designed with a modern premium UI and fully responsive layout for desktop and mobile users.
